Prerequisites:
- You need python 2.6 or 2.7 installed. 3.x will not work with these tools
- You need a web server that can serve pages from the directory that you are working in

Step 1:
configure data.cfg and console.cfg (if you need to change the time interval for the graph, that is located in globals.cfg)
Step 2:
Run processor.py data.cfg <json data file>
Step 3:
Run console.py console.cfg

Access the index.html page on localhost
